# Deviator

Консольное приложение для проверки гипотезы о нормальном распределении ошибок при автоматической упаковке продукции.

Проект реализует первый этап курсовой работы (**step1**) и позволяет:

- импортировать данные из Excel (`.xlsx`);
- вычислять статистические характеристики отклонений;
- проверять гипотезу о нормальности распределения по критерию Шапиро–Уилка;
- строить гистограмму эмпирического распределения и сравнивать её с теоретической нормальной кривой.

---

# Формат входных данных

Программа принимает Excel-файл (`.xlsx`) с **одним листом** и следующими колонками:

| id | nominal | fact |
|----|---------|------|
| 1  | 1000    | 1014 |
| 2  | 1000    | 995  |
| 3  | 1000    | 1007 |

### Описание колонок

- `id` — идентификатор записи;
- `nominal` — номинальный вес упаковки;
- `fact` — фактический вес упаковки.

---

# Что вычисляет программа

Для каждой строки считается отклонение:

```text
error = fact - nominal
```

Пример:

```text
1014 - 1000 = +14
995 - 1000 = -5
```

---

## Выходные данные

После анализа программа выводит:

### 1. Количество строк
Общее количество записей во входном Excel-файле.

---

### 2. Объем выборки
Количество рассчитанных отклонений.

---

### 3. Среднее арифметическое отклонений
Показывает среднее смещение:

```text
mean(error)
```

---

### 4. Среднее арифметическое модулей отклонений
Показывает среднюю величину ошибки без учета знака:

```text
mean(abs(error))
```

---

### 5. Дисперсия
Показывает разброс данных относительно среднего.

---

### 6. Среднеквадратичное отклонение (Sigma)
Корень из дисперсии:

```text
sigma = sqrt(variance)
```

---

### 7. Асимметрия
Показывает перекос распределения:

- `≈ 0` — распределение близко к нормальному;
- `> 0` — чаще встречается перевес;
- `< 0` — чаще встречается недовес.

---

### 8. Критерий Шапиро–Уилка
Используется для проверки гипотезы о нормальности.

Выводятся:

- `W` — статистика критерия;
- `p-value`.

Интерпретация:

- `p > 0.05` → нет оснований отвергать гипотезу о нормальности;
- `p <= 0.05` → распределение статистически отличается от нормального.

---

### 9. График распределения

Строится окно с графиком:

- гистограмма эмпирических данных;
- теоретическая кривая нормального распределения.

Это позволяет визуально сравнить реальные данные с идеальным нормальным распределением.

# Установка зависимостей

```bash
pip install pandas numpy scipy matplotlib openpyxl
```

или через `requirements.txt`:

```bash
pip install -r requirements.txt
```

---

# Запуск программы

Из корня проекта выполнить:

```bash
python main.py
```

После запуска программа попросит путь к Excel-файлу:

```text
Введите путь к Excel файлу: data.xlsx
```

Если файл находится в корне проекта:

```text
data.xlsx
```

Если в другой папке:

```text
/Users/andrey/Desktop/data.xlsx
```
